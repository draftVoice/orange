## Installation

Browser:

```html
<script src='dist/pumelo.js'></script>
```

[Node.js](http://nodejs.org):

```bash
npm install pumelo --save
```

```js
var pumelo = require('pumelo');
```

ES6 module:

```js
import pumelo from 'pumelo'
```
